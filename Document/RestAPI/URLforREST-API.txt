Below are the urls for making call to the following apis… 
 
Show Accounts API call
HTTP Method GET 
URL: http://localhost:8080/mongoDB/showAccounts

Get Account on the basis of account ID API call 
HTTP Method GET 
URL :  http://localhost:8080/mongoDB/showAccount?accountId=0019000000QoJxRAAV

Create Account (now only account name need to send) API call 
HTTP Method GET 
URL : http://localhost:8080/mongoDB/createAccounts?name=test_name

Update Account  on the basis of account ID need to send new Name and city API call 
HTTP Method GET 
URL : http://localhost:8080/mongoDB/updateAccount?accountId=0019000000QoJxRAAV&newName=test1&city=indore

Delete Account on the basis of account ID API call 
HTTP Method GET 
URL : http://localhost:8080/mongoDB/deleteAccount?accountId=0019000000QoQ6EAAV

Access user define Apex REST Resource form local API call 
HTTP Method GET 
URL : http://localhost:8080/mongoDB/googleAccountAccessToken
